# 中国免签旅游指南 - PRD（产品需求文档）

## 1. 产品概述
面向持 144 / 240 小时过境免签资格的外国游客，提供一站式「来华行前规划」静态网站。核心解决 4 个问题：**怎么来 → 怎么付 → 怎么走 → 花多少**。
- 目标用户：欧美、日韩、东南亚、俄区持过境免签资格来华游客
- 核心价值：以「工具驱动决策」（预算计算器 + 行程规划器）替代「长文阅读」
- 商业模式：Google AdSense 广告变现（通过审核为首要前提）

## 2. 核心功能

### 2.1 用户角色
- 仅匿名访客，无登录注册
- 偏好（语言、最近一次行程）持久化在 `localStorage`

| # | 页面 | 路径示例 | 主要模块 |
|---|------|----------|----------|

### 2.3 核心页面详情

#### 首页
- **Hero**：全屏原创 Canvas 背景（山水渐变 + 飘移粒子）+ 主标题 + 次标题 + 「开始规划」CTA
- **政策摘要卡**：144h / 240h 双卡片对比（小时数、覆盖区域、典型口岸）
- **4 大痛点入口**：来华 / 支付 / 交通 / 预算
- **工具入口**：2 大内嵌工具卡（计算器、规划器）
- **精选城市**：4 张城市卡（上海、北京、成都、西安）

#### 预算计算器
- **输入**：天数（1-10）、城市数（1-3）、酒店档次（经济/中端/豪华）、餐饮档次、城市内交通方式、景点门票数量、购物预算、其他
- **输出**：按城市拆分 + 分项小计 + 美元/人民币双币种总计
- **特性**：可保存为 URL 分享、可打印（Print CSS）

#### 行程规划器
- **输入**：小时数（144 / 240）、同行人数、兴趣标签（历史/美食/购物/自然/现代都市/亲子）
- **输出**：推荐 3-5 城组合 + 每日行程卡（含景点 + 用时 + 预估花费）
- **算法**：基于城市间高铁时间表 + 兴趣标签 + 最低建议停留天数，贪心生成

## 4. UI 设计

### 4.1 风格方向
**新东方极简 (Neo-Oriental Minimalism)** —— 致敬中国山水，但不落俗套。
- **主色**：`#0F1115`（墨黑）/ `#F5F0E6`（米白）
- **点缀**：`#C03B2B`（辰砂红）/ `#3F7C85`（山水青）
- **字体**：标题 `Playfair Display` + `Noto Serif SC`，正文 `Manrope` + `Noto Sans SC`
- **按钮**：圆角 6px，主按钮实色，hover 上浮 2px
- **布局**：移动端优先；桌面端 ≥1024px 进入双栏

### 4.2 动态背景（原创实现）
- **拒绝抄袭**：完全自写，无任何 tripcngo 素材
- **技术**：Canvas 2D，多层径向渐变 + 简单噪声 + 缓动漂移
- **交互**：鼠标视差（≤5px 偏移）
- **性能**：60fps，< 5% CPU，遵循 `prefers-reduced-motion`
- **资源**：零外部素材，单文件 < 50KB JS

### 4.3 响应式
- 断点：`sm 640 / md 768 / lg 1024 / xl 1280`
- 触摸：所有交互目标 ≥ 44×44px
- 字体：流体 `clamp()` 字号

## 5. AdSense 合规清单

| 要求 | 落地方式 |
|------|----------|
| 原创内容 | 全部文字原创 + 原创 SVG 插图；无复制粘贴 |
| 必要页面 | /about /contact /privacy /terms /disclaimer 全部存在且可访问 |
| 联系方式 | 真实邮箱（页面 footer + /contact） |
| 隐私政策 | 包含广告 Cookie、第三方供应商、GA4、数据保留期限 |
| Cookie 同意 | GDPR 合规的同意横幅（拒绝也可继续浏览） |
| 充足内容 | ≥ 15 个独立内容页（首页 + 5 工具页 + 5 城市页 + 4 法律页 = 至少 15 页骨架） |
| HTTPS | Cloudflare Pages 自动提供 |
| 站点地图 | `sitemap.xml` 列出全部 URL 与 hreflang |
| `robots.txt` | 允许全部，但声明 AI 训练爬虫规则 |
| 广告位 | 仅在内容流中（非弹窗、不遮挡交互元素、不诱导点击） |
| 禁止品类 | 不出现成人、赌博、药品、侵权、医疗建议等内容 |
| 多语言 | hreflang 完整声明，避免被判低质重复内容 |

## 6. 成功标准（可验证）
1. Lighthouse 性能 ≥ 90，SEO ≥ 95，可访问性 ≥ 90
2. 首屏 LCP < 2.5s，整页 < 500KB（gzipped）
3. AdSense 申请一次通过
4. 预算计算器计算误差 < 5%（对比实际携程/Booking 数据抽样 5 城）
5. 行程规划器生成路线的人工通过率 ≥ 80%
6. 10 种语言全部可切换、文本无乱码
